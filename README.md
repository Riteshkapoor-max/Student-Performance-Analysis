# Student-Performance-Analysis
A data analytics project that explores student academic performance using Python and Power BI, delivering KPIs, trends, and actionable insights through interactive dashboards.
##  Project Overview

The **Student's Performance Dashboard** is an interactive Power BI project designed to analyze student academic performance across Mathematics, Reading, and Writing subjects. The dashboard helps educators, institutions, and analysts identify factors influencing student success and uncover actionable insights for improving educational outcomes.

The analysis explores the impact of:

- Gender
- Lunch Program
- Test Preparation Course
- Parental Education Level
- Subject-wise Performance Distribution

---

##  Project Objectives

- Monitor overall student academic performance.
- Identify factors affecting student scores.
- Compare performance across demographic groups.
- Measure the effectiveness of test preparation programs.
- Provide data-driven insights for educational improvement.

##  Dashboard Insights

### 1. Gender-Based Performance
- Female students outperform male students in **Reading** and **Writing**.
- Male students perform slightly better in **Mathematics**.
- Overall academic performance is higher among female students.

### 2. Test Preparation Impact
- Students who completed the test preparation course scored significantly higher across all subjects.
- Test preparation has a positive influence on academic performance.
- Increasing participation could improve overall pass rates.

### 3. Lunch Program Impact
- Students with a **Standard Lunch** program achieved higher scores compared to those receiving **Free/Reduced Lunch**.
- This suggests a relationship between socioeconomic conditions and academic performance.

### 4. Parental Education Impact
- Students whose parents possess higher educational qualifications tend to achieve better academic results.
- Academic performance generally improves with increasing parental education levels.

### 5. Score Distribution Analysis
- Most students score between **60–79 marks**.
- Relatively few students achieve extremely low or extremely high scores.
- Student performance follows a near-normal distribution pattern.

### 6. Pass Rate Analysis
- Approximately **60% of students passed**, indicating opportunities for targeted academic interventions.
- Focus should be placed on lower-performing student segments to improve success rates.
